# aaronbailey.co

Personal portfolio site built with [Astro](https://astro.build). Deploys to GitHub Pages.

## Local development

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # production build → dist/
```

## Deployment

Pushes to `main` auto-deploy via GitHub Actions. Custom domain managed through Squarespace DNS.
